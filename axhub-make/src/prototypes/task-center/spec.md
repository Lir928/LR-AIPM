# 任务中心页面配置

## 页面信息

- **页面名称**：任务中心
- **页面路径**：/prototypes/task-center
- **页面类型**：主页面
- **描述**：任务中心页面，包含任务列表、任务统计、筛选功能和创建任务功能

## 配置选项

### 基本配置

| 配置项 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| pageTitle | input | 任务中心 | 页面标题 |
| primaryColor | colorPicker | #10b981 | 主色调 |
| secondaryColor | colorPicker | #3b82f6 | 次要色调 |

### 数据配置

| 数据项 | 类型 | 描述 |
| --- | --- | --- |
| tasks | array | 任务列表数据 |
| statistics | object | 任务统计数据 |

### 事件配置

| 事件名称 | 描述 | 参数 |
| --- | --- | --- |
| onTaskClick | 点击任务卡片时触发 | { task: Task } |
| onCreateTask | 点击创建任务按钮时触发 | {} |
| onStatusChange | 更改任务状态时触发 | { taskId: number, status: string } |
| onFilterChange | 更改筛选条件时触发 | { filter: string } |

### 动作配置

| 动作名称 | 描述 | 参数 |
| --- | --- | --- |
| refreshTasks | 刷新任务列表 | {} |
| addTask | 添加新任务 | { title: string, description: string, priority: string, dueDate: string, assignee: string } |
| updateTaskStatus | 更新任务状态 | { taskId: number, status: string } |

## 页面结构

1. **顶部导航栏**：包含页面标题、搜索框和用户头像
2. **左侧边栏**：包含任务中心、任务统计、系统设置等导航项
3. **主内容区**：
   - 任务统计概览：显示任务总数、待处理、进行中、已完成的任务数量
   - 任务筛选区：包含状态筛选按钮和创建任务按钮
   - 任务列表：表格形式展示任务，包含任务标题、优先级、状态、截止日期、负责人等信息
   - 分页控件：用于切换任务列表页码
4. **右侧边栏**：包含快速操作和最近任务

## 交互逻辑

- **任务筛选**：点击筛选按钮可以筛选不同状态的任务
- **任务点击**：点击任务标题可以查看任务详情
- **创建任务**：点击创建任务按钮可以打开创建任务表单
- **状态更新**：可以更改任务的状态
- **分页**：可以通过分页控件切换任务列表页码

## 响应式设计

- **桌面端**：完整布局，三栏结构
- **平板端**：侧边栏可折叠，两栏结构
- **移动端**：单栏结构，侧边栏通过抽屉式菜单访问

## 设计规范

- **主色调**：#10b981（绿色）、#3b82f6（蓝色）
- **辅助色**：#f3f4f6（浅灰）、#e5e7eb（中灰）、#9ca3af（深灰）
- **按钮样式**：圆角 8px，主按钮使用主色调，次要按钮使用辅助色
- **字体**：系统默认字体，标题 18-24px，正文 14-16px
- **布局**：卡片式布局，间距 16px，阴影效果轻微
- **图标**：使用 Lucide React 图标库
