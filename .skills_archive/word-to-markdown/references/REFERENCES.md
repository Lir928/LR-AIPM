# Word 转 Markdown 参考文档

## 转换对照表

### 文本样式

| Word 样式 | Markdown 语法 | 示例 |
|-----------|---------------|------|
| 标题 1 | `# 标题` | # 一级标题 |
| 标题 2 | `## 标题` | ## 二级标题 |
| 标题 3 | `### 标题` | ### 三级标题 |
| 粗体 | `**文本**` | **粗体** |
| 斜体 | `*文本*` | *斜体* |
| 下划线 | `__文本__` | __下划线__ |
| 删除线 | `~~文本~~` | ~~删除线~~ |
| 代码 | `` `代码` `` | `code` |

### 列表

**无序列表** (bullet points)：
```markdown
- 项目 1
- 项目 2
  - 子项目
```

**有序列表**：
```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A1  | B1  | C1  |
| A2  | B2  | C2  |
```

### 图片引用

```markdown
![图片描述](filename_media/image_001.png)
```

### 超链接

```markdown
[链接文字](https://example.com)
```

## 图片处理说明

1. 图片会从 Word 文档中提取
2. 图片文件保存在 `{filename}_media/` 文件夹
3. Markdown 中使用相对路径引用图片
4. 支持的格式：PNG、JPG、GIF、WEBP

## 限制说明

- Word 中的字体颜色会丢失（Markdown 不支持颜色）
- 复杂的表格合并可能显示不正确
- 页眉、页脚、脚注不会转换
- 文本框内容可能不转换
