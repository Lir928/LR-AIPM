# 事件中心 Axhub Make Prompt

## 项目概述

创建一个现代化、响应式的事件中心平台，包含事件管理、通知、分析和统计功能。

## 设计风格

- **主色调**：#1E88E5（蓝色），代表专业、可信赖
- **辅助色**：
  - 成功：#4CAF50（绿色）
  - 错误：#F44336（红色）
  - 警告：#FFC107（黄色）
- **按钮风格**：圆角按钮，有明显的悬停效果
- **字体**：系统默认无衬线字体，标题 16-20px，正文 14px，辅助文字 12px
- **布局风格**：响应式布局，左侧固定导航栏，右侧内容区域
- **图标风格**：使用 Material Design 图标，简洁明了

## 页面结构

### 1. 事件列表页

**布局**：
- 左侧导航栏：包含事件中心、事件分析、通知设置等菜单
- 顶部筛选栏：包含事件类型、状态、时间范围筛选器和搜索框
- 主体内容区：事件表格，包含复选框、事件标题、类型、优先级、状态、创建时间、操作按钮
- 底部：分页控件

**功能**：
- 支持按事件类型、状态、时间范围筛选
- 支持按创建时间、优先级、状态排序
- 支持按关键词搜索
- 支持批量操作和单个事件操作
- 响应式设计，在移动端转为卡片式布局

### 2. 事件详情页

**布局**：
- 左侧导航栏
- 顶部：事件标题和操作按钮（编辑、删除、关闭）
- 主体内容区：
  - 基本信息卡片：展示事件标题、描述、类型、优先级、状态等
  - 历史记录时间线：按时间顺序展示事件的历史操作
  - 评论区：评论输入框和评论列表
  - 关联资源：相关文档和链接

**功能**：
- 支持编辑事件基本信息
- 支持添加评论和回复
- 支持上传和查看关联资源
- 支持状态变更和历史记录查看

### 3. 事件创建页

**布局**：
- 左侧导航栏
- 顶部：页面标题"创建事件"
- 主体内容区：
  - 事件表单：包含标题、描述、类型、优先级、负责人等字段
  - 附件上传区域：支持拖拽上传

**功能**：
- 表单验证
- 附件上传
- 提交创建事件

### 4. 事件分析页

**布局**：
- 左侧导航栏
- 顶部：页面标题"事件分析"
- 主体内容区：
  - 统计卡片：显示事件总数、待处理数、已完成数等关键指标
  - 图表区域：包含事件类型分布饼图、状态分布柱状图、时间趋势折线图
  - 导出按钮：支持导出统计数据

**功能**：
- 数据可视化展示
- 支持时间范围选择
- 支持数据导出

### 5. 通知设置页

**布局**：
- 左侧导航栏
- 顶部：页面标题"通知设置"
- 主体内容区：
  - 规则列表：显示现有通知规则
  - 规则编辑表单：用于创建和编辑通知规则

**功能**：
- 创建新通知规则
- 编辑现有规则
- 删除规则
- 配置通知方式和接收人

## 交互设计

- **悬停效果**：按钮、卡片等元素有明显的悬停效果
- **加载状态**：数据加载时显示加载动画
- **错误提示**：操作失败时显示错误提示
- **成功提示**：操作成功时显示成功提示
- **表单验证**：实时表单验证和错误提示
- **分页交互**：点击页码或上一页/下一页按钮切换数据
- **筛选交互**：选择筛选条件后自动刷新数据
- **搜索交互**：输入关键词后按回车或点击搜索按钮执行搜索

## 响应式设计

- **桌面端**（>1200px）：完整展示所有功能和布局
- **平板端**（768px-1200px）：导航栏可折叠，表格适应宽度
- **移动端**（<768px）：导航栏转为底部或顶部菜单，表格转为卡片式布局

## 组件要求

- **按钮**：主按钮、次要按钮、文本按钮、图标按钮
- **表单元素**：输入框、文本域、下拉选择框、日期选择器、复选框、单选按钮
- **卡片**：信息卡片、统计卡片
- **表格**：支持排序、筛选、分页
- **图表**：饼图、柱状图、折线图
- **时间线**：垂直时间线，展示历史记录
- **评论区**：支持评论和回复
- **上传组件**：支持拖拽上传和文件选择
- **导航栏**：左侧固定导航，支持折叠
- **面包屑**：显示当前页面路径
- **模态框**：用于编辑、删除确认等操作

## 页面切换

- 从事件列表页点击事件标题或查看按钮进入事件详情页
- 从事件列表页点击创建按钮进入事件创建页
- 从导航栏点击事件分析进入事件分析页
- 从导航栏点击通知设置进入通知设置页
- 从事件详情页点击编辑按钮进入编辑页面
